Quoi de neuf dans ArtOS Antelope Canyon

ArtOS Antelope Canyon offre une nouvelle façon de travailler, de jouer ou de faire ce que l'on veut. ArtOS intègre Windows 10 pour utiliser vos applications Windows préférées.

Lire les changements

Travailer ou jouer.

Restez productif avec la vue multitâche, Multi-Desk, Ne pas déranger, et plus encore.

Vue Multitâches

Les espaces de travail permettent d'organiser votre travail par tâche. Gardez le travail et les loisirs séparés, mais à portée d'un glissement ou d'un tapotement.

Multi-Desk

AArtOS Multi-Desk offre une nouvelle façon d'organiser vos fichiers et vos applications. Vous pouvez ajouter 2 "Bureaux" pour chaque utilisateur, chaque mode de Multi-Desk fournit de nouvelles applications par défaut, des options et beaucoup plus.

Image Dans l'Image

YVous voulez travailler et aussi regarder une vidéo ? Grâce à l'image dans l'image, c'est possible. Quand vous le souhaitez, passez la souris sur l'icône PiP de la vidéo pour l'active

Ne Pas Déranger

Ne pas déranger organise vos notifications pour vous. paramétrez les notifications des applications pour ne conserver que les notifications importantes

Apps incluses, sans celles que vous ne voulez pas.

ArtOS inclut ces applications, vous pouvez les désinstaller, nous détestons les bloatware. Vous pouvez voir les captures d'écran et les caractéristiques des applications.

Musique

Organisez et écoutez votre musique. Naviguez par album, créez des listes de lecture et intégrez Spotify pour une meilleure expérience musicale.

Navigate

Surfez sur le Web plus vite que jamais. Regardez des vidéos, lisez ou faites tout ce que vous pouvez dans un navigateur sécurisé. Une édition pour développeurs est disponible pour créer des sites web et ajoute les fonctionnalités avancées pour les développeurs.

Mail

A simple mail app

Photos

Ajoutez vos photos de famille et modifiez-les. Vous pouvez créer des groupes pour organiser vos photos.

Videos

Toutes personne qui regarde des vidéos ou des films doit avoir cette application. YouTube est intégré pour regarder les vidéos de vos créateurs préférés. Basé sur VLC

Calendar

Créez et visualisez des événements. Synchronisé avec vos contacts pour ajouter automatiquement les anniversaires. Et ce que l'on attend d'un calendrier.

Fichiers

Un explorateur de fichiers avec des onglets (pas comme Windows 11). Une IA suggère des fichiers en fonction de votre utilisation

Terminal

AFonctions avancées d'ArtOS, peut exécuter des scripts et des programmes. Uniquement pour les utilisateurs avancés

MakeAnApp

MMakeAnApp est l'IDE parfait. Il intègre Git, une version de développement d'ArtOS, et tous les outils qu'un développeur veut pour faire des applications.

Virtualizer

Ici le Notepad Windows (notepad.exe) est en cours d'execution

ArtOS intègre une installation Windows (nécessite une clé de produit Windows) pour utiliser vos applications Windows préférées.

Big compatibility

Virtualizer est compatible avec une grande partie des apps.

Optimisé pour les jeux vidéos

Virtualizer est optimisé pour les jeux. Virtualizer détecte et adapte automatiquement les performances si vous utilisez une application ou un jeu.

Gérer les applications Windows

Vous pouvez télécharger des applications à partir d'Internet et les installer de la manière habituelle. Elles s'installeront automatiquement dans les programmes de Windows et vous pourrez gérer les applications installées directement dans Virtualizer.

Fait pour les utilisateurs

ArtOS est conçu pour les utilisateurs, ce ne devrait pas être les utilisateurs qui doivent s'adapter aux interphases. ArtOS répond à ce problème, ArtOS est entièrement personnalisable avec des thèmes et des packs d'icônes.

L'ami des développeurs pour être l'ami des utilisateurs

MakeAnApp fournit tous les outils nécessaires aux développeurs pour créer vos applications. Combiné avec ArtOS DevKit, Les developeurs font des applications en C# grâce a Microsoft DotNet.

Respect de la vie privée. De bout en bout.

Vos données n'appartiendront toujours qu'à vous. Avec Files Restore And Rewind vous n'allez jamais perdre un fichier. Tous vos fichiers sont sauvegardés sur le disque de l'ordinateur, pas dans un Cloud.

(Nos critéres de sécuritée)

Anti-espion

ArtOS vous aide à protéger votre vie privée. Lorsqu'une application utilise votre microphone, un indicateur s'affiche. La première fois qu'une application veut utiliser votre microphone, une notification d'autorisation s'affiche.

Permissions

Gérez les permissions des fichiers, dossiers et des disques. Vous pouvez également définir des autorisations pour les fichiers partagés.

Maniaque de l'espace

ArtOS supprime automatiquement les fichiers inutilisés et les fichiers temporaires, ce qui vous permet d'avoir beaucoup de place pour vos applications et vos fichiers.
